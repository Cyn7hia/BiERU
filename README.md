# BiERU
This is the code for BiERU (GPU version)

## requirements

* python version == 3.7.3
* pytorch == 1.4.0
* numpy ==1.18.1
* scipy == 1.4.1
* pandas == 1.0.3
* sklearn == 0.23.1

For those who want to use the MELD and AVEC datasets, please refer to the following google drive link.
